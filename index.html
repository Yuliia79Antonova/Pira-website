Short Character Stories

Animation project by Yuliia Antonova — Interactive Digital Art & Design. A series of short emotional animations designed for social media storytelling.

About The Project

This project explores short-form character animation designed for modern social media platforms such as TikTok and Instagram Reels. Each animation lasts around 10–15 seconds and focuses on visual storytelling through expressive character animation. The project demonstrates how simple narrative moments and strong character performance can communicate emotion effectively within a very short duration. All animations were created using Adobe After Effects with assets prepared in Photoshop.
